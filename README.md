✅ Key Features

Safe for all deployments — does not delete or change any local files or Azure resources.

Handles all resources: supported, unsupported, and cyclic references.

Generates placeholder Bicep for resources that cannot decompile.

Detects added, changed, and removed resources.

Generates drift summary (drift-summary.txt) with:

Added resources

Changed resources

Unsupported resources

Commands to inspect manually (az resource list ...)

Reusable for any Resource Group — just provide -ResourceGroup and -LocalRoot.
